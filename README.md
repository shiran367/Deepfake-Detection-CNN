# Deepfake-Detection-CNN
Deepfake detection model using Convolutional Neural Networks (CNN) and Python. Developed as part of Computer Science studies at Ruppin Academic Center.


## Project Overview
This project addresses the growing challenge of **Deepfakes**—AI-generated synthetic media. The goal was to develop a high-accuracy binary classification model capable of distinguishing between real human faces and synthetic faces generated by neural networks.

The project demonstrates a professional evolution from building a baseline **CNN** from scratch to implementing advanced **Transfer Learning** techniques using **MobileNetV2**.

## Key Results 
* **Baseline CNN Accuracy:** ~84%.
* **Transfer Learning (MobileNetV2) Accuracy:** **94%**.
* **Key Insight:** Utilizing pre-trained weights from ImageNet significantly improved the model's ability to generalize and identify subtle facial textures characteristic of synthetic images.

## Technical Stack 
* **Language:** Python.
* **Framework:** TensorFlow / Keras.
* **Deep Learning Architectures:** CNN (Custom), MobileNetV2 (Transfer Learning).
* **Data Processing:** OpenCV, NumPy, Pandas.
* **Tools:** Matplotlib, Seaborn, Scikit-learn (Metrics & Reporting).

## Methodology 

### 1. Data Preparation
* Used the **"Deepfake and Real Images"** dataset from Kaggle.
* Data split: **60% Training, 20% Validation, 20% Testing**.
* **Data Augmentation:** Applied rotation, flipping, and zooming to enhance model robustness and prevent overfitting.

### 2. Model Evolution
* **Custom CNN:** Built a multi-layer Convolutional Neural Network with Batch Normalization and L2 Regularization to establish a baseline.
* **Transfer Learning:** Integrated **MobileNetV2**, freezing initial layers to preserve low-level feature extraction capabilities (edges, textures).
* **Fine-Tuning:** Unfroze the top layers (100-155) with a very low learning rate (0.00001) to adapt the model specifically to facial artifacts in Deepfakes.

### 3. Visualization & Explainability
* Implemented **Grad-CAM** to visualize "Focus Areas"—highlighting which parts of the face the model used to make its decision (Real vs. Fake).

## How to Run
1. Clone the repository.
2. Ensure you have the required libraries installed: `pip install tensorflow opencv-python matplotlib seaborn scikit-learn`.
3. Open `DeepfakeVSRealIMG.ipynb` in Google Colab or Jupyter Notebook.
4. Run the cells to see the training process, evaluation metrics, and Grad-CAM visualizations.
